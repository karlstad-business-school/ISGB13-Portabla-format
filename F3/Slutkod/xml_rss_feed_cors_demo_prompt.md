# Context
XML RSS feed CORS demo för undervisning.

## Teknisk kontext
- Programmeringsspråk: JavaScript
- CSS-ramverk: Tailwind-klasser
- Använd inte template strings när du skriver JavaScript
- Spara och uppdatera filerna du jobbar i. 

## Din uppgift
Skriv i filen index.html och skriv HTML-kod, ingen JavaScript eller CSS.

### Krav på design och struktur
- Designa med Tailwind-klasser en clean hemsida för att visa upp xkcd.com-comics
- Designen ska vara väldigt enkel men snygg
- Innehållet ska vara centrerat på sidan och inte för brett
- Skapa en hero-banner med en passande titel
- Använd färgerna: mörkblått, navy eller indigo, vitt och svart
- Sidan ska vara clean, vertikal och gå att scrolla igenom för flera comics
- Använd ingen egen CSS, bara Tailwind-klasser
- I HTML koden ska DIV:en som ska hålla innehållet från xkcd.com-comics ha id="feed"

## Viktigt
- Skriv endast HTML
- Ingen JavaScript-kod
- Ingen vanlig CSS
- Håll koden tydlig och enkel
